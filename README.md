# DeadlineZero - AI Productivity Agent

An autonomous AI productivity agent built for Vibe2Ship Hackathon by Coding Ninjas x Google for Developers.

## Problem Statement
The Last-Minute Life Saver - Students, professionals, and entrepreneurs frequently miss deadlines and important commitments.

## Solution
DeadlineZero is an AI agent that proactively helps users plan, prioritize, and complete tasks before deadlines are missed.

## Features
- Voice and text task input
- AI-powered subtask breakdown using Gemini 2.0 Flash
- Google Calendar auto-scheduling
- Smart context-aware reminders
- Priority scoring system
- Progress tracking dashboard
- Edit and delete tasks
- Mark subtasks as complete
- Auto-rescheduling of missed tasks

## Tech Stack
- HTML, CSS, JavaScript
- Tailwind CSS
- Gemini 2.0 Flash API
- Google Calendar API
- Web Speech API
- Google Cloud Platform

## Google Technologies Used
